# Факториал

В математике, факториал множества неотрицательных целых чисел `n`,
обозначается `n!`, является произведением всех натуральных чисел 
меньше или равным `n`. Например:

```
5! = 5 * 4 * 3 * 2 * 1 = 120
```

| n     | n!                          | 
| ----- | --------------------------: |
| 0     | 1                           |
| 1     | 1                           |
| 2     | 2                           |
| 3     | 6                           |
| 4     | 24                          |
| 5     | 120                         |
